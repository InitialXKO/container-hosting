Help I got an error

When doing `docker-compose up` I see: IndexError: list index out of range ```` File "compose/service.py", line 1950, in build IndexError: list index out of range ```
  • Did you recently try to build docker containers on an M1 mac? Do you usually use `--load`?
  • Try `COMPOSE_DOCKER_CLI_BUILD=0 docker-compose up`
  • See Failed to execute script docker-compose. IndexError: list index out of range
ERROR: Encountered errors while bringing up the project

If you see the bind port is already allocated error, this usually means you have another application running on the same port (or forgot to stop docker-compose last time.

Stop the other running application and start again
